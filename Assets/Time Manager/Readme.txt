Press "Tab" to reset scene
Hold "Enter" to rewind time
Press "Space" to slow down time
Press "F" to fast forward time
Press "Escape" to pause/ resume game

* Simply drag and drop any GameObject into the objects to rewind array, the script will keep track of the 
   position, rotation and scale of the game object and apply rewind if input is pressed.
* The slow motion, fst forward and pause attributes affects all gameobjects in the scene.
